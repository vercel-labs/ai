# `appendResponseMessages()`

Appends an array of CoreMessage objects (from the AI response) to an existing array of UI messages.
It generates new IDs and timestamps for each appended message and merges tool-call results with the previous assistant message. Useful for maintaining a unified message history when working with AI responses in a client-side chat application.
